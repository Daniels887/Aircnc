## Aircnc - Desenvolvido o back-end e o mobile da semana omnistack da Rocketseat

### Back-end 

- Express
- Mongoose
- Multer

Para rodar o Backend
- Npm install ou Yarn para instalar as dependências
- Npm run dev ou yarn dev
